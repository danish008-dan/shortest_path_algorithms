Graph Algorithms Repository
===========================

Algorithms Included
-------------------
1. Dijkstra – Shortest path (non-negative weights)
2. Bellman-Ford – Shortest path (negative weights + cycle detection)
3. Prim's – Minimum Spanning Tree (Greedy)
4. Kruskal – Minimum Spanning Tree (DSU based)
5. Topological Sort – DAG ordering

Key Concepts
------------
- Greedy Algorithms
- Priority Queue
- Disjoint Set Union (Union-Find)
- Directed Acyclic Graphs (DAG)

Complexities
------------
Dijkstra: O(E log V)
Bellman-Ford: O(VE)
Prim’s: O(E log V)
Kruskal: O(E log E)
Topological Sort: O(V + E)

Use Cases
---------
- Network routing
- Task scheduling
- Dependency resolution
- Infrastructure design
